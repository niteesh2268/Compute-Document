Parameters

  • name – Unique name of the feature group

  • entities – List of Strings representing the entities of the feature group

  • sources – list of sources in the feature group

  • schema – List of Features which are part of this feature group.

  • sink – Sink for the feature group. If not defined a new S3 sink is generated

  • compute_meta_data – compute details associated with the feature group

  • user – user who is owner of the feature group

  • tags – tags associated with the feature group

  • description – tags associated with the feature group

  • notification_channel – notification_channel associated with the feature group

  • status – whether a feature group is active or not

Returns

update_feature_group(name: str, entities: List[str], sources: List[Source], schema: List[Feature], sink: Optional[Sink], compute_meta_data: ComputeMetaData, tags: List[str], description: str, user: str, notification_channel: List[str], status: Literal['ACTIVE', 'INACTIVE']) → object

Update a feature group. There is versioning for feature groups which is maintained at FCT. If a feature group does not exist with the same name then an error is thrown. You can create a new FG using register_feature_group_method.

If a feature group exists with the same name then there can be 2 scenarios:

  1. A new version is created if any one of [schema, sources, entities, sinks] is different

  2. Else, the existing feature group is updated

Parameters

  • name – Unique name of the feature group

  • entities – List of Strings representing the entities of the feature group

  • sources – list of sources in the feature group

  • schema – List of Features which are part of this feature group.

  • sink – Sink for the feature group. If not defined a new S3 sink is generated

  • compute_meta_data – compute details associated with the feature group

  • user – user who is owner of the feature group

  • tags – tags associated with the feature group

  • description – tags associated with the feature group

  • notification_channel – notification_channel associated with the feature group

  • status – whether a feature group is active or not

Returns

Feature Group Object which has feature group definition and store details
